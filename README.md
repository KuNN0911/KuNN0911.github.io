# HenryKu.github.io

Henry Ku 的 GitHub Pages 個人網站。

## 本機預覽

直接用瀏覽器開啟 `index.html` 即可預覽。

## 部署到 GitHub Pages

如果這個 repo 要當作專案頁面，推上 GitHub 後到：

`Settings` → `Pages` → `Build and deployment` → `Deploy from a branch`

選擇 `main` branch 與 `/root` 目錄。

目前遠端 repo 是：

`https://github.com/KuNN0911/HenryKu.github.io`

啟用 Pages 後，專案頁網址通常會是：

`https://kunn0911.github.io/HenryKu.github.io/`

如果想要使用帳號首頁網址 `https://kunn0911.github.io/`，GitHub repo 名稱需要改成 `KuNN0911.github.io`。
