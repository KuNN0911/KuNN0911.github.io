# KuNN0911.github.io

Henry Ku's GitHub Pages personal website.

## Local Preview

Open `index.html` directly in a browser.

## GitHub Pages Setup

To use the account-level GitHub Pages URL:

`https://kunn0911.github.io/`

rename this repository to:

`KuNN0911.github.io`

Then go to:

`Settings` -> `Pages` -> `Build and deployment` -> `Deploy from a branch`

Select the `main` branch and `/root` folder.

After GitHub finishes publishing, the site should be available at:

`https://kunn0911.github.io/`
